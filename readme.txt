The Ridime wordpress plugin allows you to earn crypto coupons for your consumers through the Woo commerce ecommerce extension.
  
== Description ==
  
The plugin works simply:

1. Create your account on Ridime and create your first campaign.
2. Get your API key automatically created in the name of your brand.
3. Download the WordPress plugin and install it.
4. Download the documentation for more information.

  
== Installation ==
  
1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Extract the zipped file and upload the folder crypto-coupon-redeem to /wp-content/plugins/ directory.
